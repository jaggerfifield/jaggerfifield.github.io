Hello world! This is my personal website, where I try to talk about some projects or interesting things that are happening. I am not really a web designer but wanted to try putting together some html and see how it goes. All of the pages have been hand programmed by myself, with a bit of help from stackoverflow or w3School. Since my site is hosted by github pages, I am unable to access the html server but if I could I would be experimenting with HTML generation server-side, so the site would be built as you used it. But here, this is not the case. Instead, my pages are made by combining multiple fragmented html files to create a complete htmp page. The main fragments that are joined together are the header, content, and footer but it really depends on the page's content. Take this page for example, it is a combination of 3 fragments: header.html + jnet.html + footer.html -> jnet.html. You can see the page generation script here. Its just a simple cat of the seperate fragments. The reason I do this is (again im not a web programmer) to have a single file control site wide changes. So when I edit header.html it applys the changes to evey webpage that used the header. I would hope there is a better way to do this but could not figure it out, so this is what im going to use for now. Anyway Im suprised that you are still reading this, are you really that interested in a random website made by a non-web programmer?
I played around with some different styles when making the site, you can see the most early design here or another one here. You can really see what design elements were carried on to the next interation, like how the header has the centered text with a plane below for navigation. Currently the theme of the site is following the nord color scheme. I am enjoying this theme so it will probably stay around for now. One design choice I made, is to avoid pictues on my site. This is to hopefully reduce network traffic, so the pages are only downloading text html and not graphics. Its not a forced rule and you may find some graphics across the site, but im trying to keep it down to a minimum. It could also be a fun game, have you found a page with a image on it yet? I belive that sometimes a image is needed to convey inforamtion so Im not going to ban them from my site, but who doesnt want a page to load faster.
